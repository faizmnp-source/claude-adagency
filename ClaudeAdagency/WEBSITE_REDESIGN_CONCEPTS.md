# TheCraftStudios — 10 New Full-Website Design Directions

This document proposes 10 complete visual directions for the entire website (marketing + product pages), while keeping existing functionality and flows unchanged.

---

## Shared non-negotiables (for all concepts)
- Keep current routes and product capabilities intact:
  - Home, Services, Pricing, Contact, Login
  - Studio, Studio Credits, Studio Calendar
- Preserve current studio logic and controls:
  - Content type selection (video/image), modes (express/auto/manual/notebook), upload/settings flow, progress pipeline, results + post actions, credits visibility.
- Redesign is visual/UX hierarchy only (no feature removal).

---

## 1) Crimson Nebula 3D
**Vibe:** Cinematic, bold, premium, futuristic.

**Visual language**
- Deep space backdrop + volumetric red glows
- Layered glass cards with edge highlights
- Metallic gradient accents on CTAs

**Palette**
- Background: `#05070F`
- Surface: `#0E1424`
- Primary: `#E50914`
- Accent: `#7C3AED`
- Text-muted: `#9CA3AF`

**Typography**
- Display: Sora/Bebas Neue hybrid strategy
- Body: Inter

**Studio layout**
- Left: vertical workflow rail
- Center: generator canvas + settings
- Right: live preview + status + actions
