# Canva Creative Direction Brief — TheCraftStudios Full Website Redesign

## 1) Project context (what this product is)
TheCraftStudios is an AI-powered digital platform focused on helping businesses create social-media-ready content (especially reels), while also offering web/app/software/branding services. The core product is an authenticated AI Studio where users upload images, configure generation settings, and produce/post content.

## 2) Current design snapshot (for reference, not for copying)
- Current visual direction is dark "space" UI with deep navy backgrounds and red accent.
- Existing primary colors:
  - Deep background: `#050B18`
  - Card backgrounds: `#0D1628`, `#0A1020`
  - Primary accent red: `#E50914`
  - Secondary text/muted: `#94A3B8`
- Typography currently mixes Bebas Neue (display) + Space Grotesk + Inter.
- Hero style is cinematic and bold, with strong uppercase headlines and glow effects.
- Overall feel today: energetic, dark, tech-agency; however UI consistency and premium polish can be significantly elevated.

## 3) Mandatory constraint — do NOT change product functionality
Canva must redesign visuals and UX presentation only.
Keep all existing features, flows, and page intents intact. No feature removals.

### Functional elements that must remain present
1. Public marketing pages:
   - Home
   - Services (and service detail pages)
   - Pricing
   - Contact
2. Auth pages:
   - Login
3. Studio system pages:
   - Studio main generation flow
   - Credits / billing page
   - Calendar / auto-pilot planning page
4. Core studio capabilities to preserve in UX:
   - Content type selection (video/image)
   - Upload area and generation settings
   - Modes (express/auto/manual/notebook)
   - Duration/package/model selections
   - Pipeline/progress states
   - Result preview with tabs and post actions
   - Instagram connect / posting actions
   - Credits awareness and purchase path

## 4) New design target (what we want)
Create an **ultimate premium 3D social-tech brand experience** that blends:
- The emotional pull of top social media products
- The trust and precision of a top enterprise AI platform
- The aspirational look of a high-end tech company

### Design keywords
Premium, cinematic, 3D depth, glassmorphism 2.0, neon restraint, spatial layering, social-native, creator-centric, AI-forward, futuristic but usable, conversion-optimized.

## 5) Brand identity direction for Canva (logo + system)
### 5.1 Logo redesign request
Design a new brand system for **TheCraftStudios** with:
- Primary logo mark (icon)
- Horizontal lockup
- Monochrome variants (light/dark)
- App/favicon version

### 5.2 Logo style guidance
- Should communicate: creativity + AI intelligence + motion/content + premium trust
- Avoid cliché “generic AI brain” symbols
- Prefer geometric form with subtle motion implication (ribbon, lens, portal, spark path, waveform-in-frame, etc.)
- Must work small (favicon) and large (hero)

### 5.3 Suggested palette evolution (Canva can refine)
- Keep legacy DNA by retaining a red-led accent family, but elevate with luxury contrast:
  - Core dark: near-black blue / charcoal
  - Primary electric red/crimson accent
  - Supporting accents: cobalt/cyan/violet highlights (subtle)
  - Neutral system for cards, borders, text states

## 6) Full website redesign scope (all pages)
Canva should provide high-fidelity designs for:
1. Homepage (complete)
2. Services index + 1 detail template reusable for all services
3. Pricing page
4. Contact page
5. Login page
6. Studio main page (complete, major focus)
7. Studio credits page
8. Studio calendar page
9. Reusable components library + design tokens

## 7) Studio page redesign — highest priority
This page should feel world-class and become the “showpiece” of the brand.

### 7.1 Experience goals
- Instantly communicates power + simplicity
- Makes users excited to create
- Feels like a premium creator cockpit
- Retains all existing controls and states, but with better hierarchy and micro-interactions

### 7.2 Layout direction (desktop)
- Left: workflow rail / step navigator
- Center: active configuration canvas + upload/generation controls
- Right: live preview, progress status, tips, and output actions
- Sticky top context bar: credits, account, notifications, IG connection state

### 7.3 UX behavior requests
- Strong state design for:
  - idle / uploading / generating / success / error
- Beautiful progress visualization for pipeline stages
- Better visual grouping of advanced settings (progressive disclosure)
- Clear, premium CTA hierarchy (Generate, Regenerate, Approve & Post)

### 7.4 Visual treatment
- Layered glass cards with depth shadows and luminous edges
- 3D/gradient mesh background with restrained motion
- Subtle parallax/lighting cues (design intent only; engineering will implement later)
- Social-media-native preview cards for reels/images
- Premium charts/stats styling for credits and usage

## 8) Information architecture and consistency rules
- Keep navigation structure and route intent unchanged.
- Unify spacing, border radius, and elevation across all pages.
- Build one coherent component language across marketing + product surfaces.
- Mobile-first responsiveness required for every redesigned screen.

## 9) Deliverables Canva should provide
1. Brand direction board (2–3 visual routes)
2. Final selected route applied to full UI kit
3. Complete high-fidelity desktop + mobile screens for all pages listed
4. Component library (buttons, inputs, cards, nav, tabs, badges, modals, progress)
5. Design tokens:
   - Colors
   - Typography scale
   - Spacing scale
   - Radius
   - Shadows
   - Motion timing guidelines
6. Export package for engineering handoff:
   - Annotated layouts
   - Interaction notes
   - Asset pack (logo SVG/PNG, icons, illustrations)

## 10) Conversion and business goals
The redesign should make visitors say:
> “I want this exact level of website and product experience for my own company.”

Primary business outcomes:
- Increase trust and perceived product quality
- Increase demo/contact conversions
- Increase studio signups and first generation completions
- Improve shareability and word-of-mouth

## 11) Copy tone direction
Confident, modern, high-agency, creator-first, concise.
Avoid overly technical jargon in hero sections; keep technical depth in expandable/product sections.

## 12) Handoff instruction for next phase (engineering)
Once design is finalized in Canva:
- We will provide the approved design system and screens to Codex.
- Codex should then implement the redesign in frontend and backend presentation layers **without changing existing business functionality/APIs**.
